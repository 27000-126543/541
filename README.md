# 大型综合医药电商与连锁药店管理系统

## 项目概述

围绕药品全生命周期，覆盖用户购药与门店运营需求的综合医药电商平台。

## 技术架构

- **后端服务**: Node.js + Express + MongoDB
- **用户端小程序**: Taro 框架（支持微信/支付宝/H5多端）
- **管理后台**: React + Ant Design Pro
- **数据存储**: MongoDB + Redis

## 项目结构

```
.
├── packages/
│   ├── server/          # 后端API服务
│   ├── admin-web/       # 管理后台Web端
│   └── mini-app/        # 用户端小程序
└── README.md
```

## 核心功能模块

### 用户端（小程序）
- 家庭健康档案管理
- 智能药品推荐
- 在线问诊与电子处方
- 药品商城（OTC/医疗器械/保健品）
- 促销活动（满减/拼团/限时秒杀）
- 医保线上支付
- 会员体系
- 订单管理（配送/自取）

### 管理后台
- 药品管理（入库/效期预警/库存盘点）
- 处方审核
- 订单管理
- 会员管理
- 门店管理
- 数据看板
- 报表导出

### 智能分析
- 热销品类预测
- 流行病趋势分析
- 库存优化建议
- 促销策略建议

## 快速开始

```bash
# 安装依赖
npm install

# 启动后端服务
npm run dev:server

# 启动管理后台
npm run dev:admin

# 启动小程序（H5模式）
npm run dev:mini
```
